Grid auto rows and grid auto columns

In CSS Grid, the grid-auto-rows and grid-auto-columns properties are used to define the size of implicitly created rows and columns in a grid when the number of rows or columns is not explicitly specified in the grid definition. These properties apply to the rows and columns that are created automatically to accommodate additional content.

Here are examples demonstrating the use of grid-auto-rows and grid-auto-columns:

Example 1: grid-auto-rows

1

4

In this example, we've explicitly defined two columns, but we haven't specified the number of rows. The grid-auto-rows: 50px; property ensures that any additional rows created implicitly will have a height of 50 pixels.
